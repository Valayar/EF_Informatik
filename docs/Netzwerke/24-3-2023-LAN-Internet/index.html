Switsches Regulieren Datenverkehr zwischen verschiedenen Hosts innnerhalb einer Netzwerkes. Router regulieren den Datenverkehr zwischen verschiedenen Netzwerken. Ein Beispiel für einen Router wäre der WLAN Kasten, welcher bei jedem Zuhause steht.
